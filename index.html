Dateien durchsuchen

Alle geteilten Materialien erscheinen unten in der Vorschau. Klicke eine Datei an, um sie zu öffnen.

So lädst du eine Datei hoch

  1. Klick auf "Drive öffnen & hochladen" oben links
  2. Google Drive öffnet sich in einem neuen Tab
  3. Klick oben links auf "+ Neu" → "Datei hochladen"
  4. Wähle deine Datei aus — fertig!
